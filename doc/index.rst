########################
lsst.ts.MTDomeTrajectory
########################

ts_MTDomeTrajectory contains the `MTDomeTrajectory` CSC and support code,
including a simple `MockDome` CSC for unit testing.

.. _lsst.ts.MTDomeTrajectory-using:

Using lsst.ts.MTDomeTrajectory
==============================

You can setup and build this package using eups and sconsUtils.
After setting up the package you can build it and run unit tests by typing ``scons``.
Building it constructs a version.py file and runs unit tests.

To run the `MTDomeTrajectory` CSC type ``runMTDomeTrajectory.py``
